# 项目简介

## 核心功能
- 管理员身份验证与权限管理
- 用户车位使用权验证
- 停车位导览图管理
- 车位提供者信息管理
  - 车位使用权证明上传
  - 车位位置指定
  - 车位闲置时间设置
  - 车位价格设置
- 车位使用者信息管理
  - 空闲车位显示
  - 车位使用权限购买
  - 订单取消功能

## 适用场景
- 小区停车资源优化配置
- 提高车位利用率
- 方便车位供需双方的信息交流

## 技术特点
- 基于Java开发，稳定可靠
- 小程序端用户交互便捷
- 数据库管理高效
- 功能模块完善，覆盖车位供需全流程
